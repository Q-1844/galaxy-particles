# GitHub Pages 部署指南

## 📋 部署前准备清单

### 1. 复制配置文件到项目根目录

将 `gi` 文件夹中的以下文件复制到项目根目录：

```bash
# 复制 .gitignore
cp gi/.gitignore ./

# 复制 .github 文件夹
cp -r gi/.github ./
```

### 2. 修改项目配置文件

#### 修改 `jiemian/vite.config.ts`
将以下内容替换到 `jiemian/vite.config.ts` 文件：

```typescript
import { defineConfig } from 'vite'
import path from 'path'
import tailwindcss from '@tailwindcss/vite'
import react from '@vitejs/plugin-react'

export default defineConfig({
  plugins: [
    react(),
    tailwindcss(),
  ],
  resolve: {
    alias: {
      '@': path.resolve(__dirname, './src'),
    },
  },
  assetsInclude: ['**/*.svg', '**/*.csv'],
  base: './',
  build: {
    outDir: 'dist',
    assetsDir: 'assets',
  },
  server: {
    fs: {
      allow: ['..'],
    },
  },
})
```

#### 修改 `jiemian/src/app/pages/Galaxy.tsx`
将 iframe 的 src 从 `/parent/galaxy.html` 改为 `./galaxy.html`：

```typescript
<iframe
  ref={iframeRef}
  src="./galaxy.html"
  className="absolute inset-0 w-full h-full border-0"
  title="银河粒子3D场景"
/>
```

### 3. 初始化 Git 仓库（如果还没有）

```bash
# 进入项目根目录
cd "c:\Users\曹文淇\Desktop\Kty - 副本"

# 初始化 Git 仓库
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "准备部署到 GitHub Pages"
```

### 4. 创建 GitHub 仓库并推送

1. 访问 https://github.com/new 创建新仓库
2. 仓库名称建议：`galaxy-particles` 或 `birthday-galaxy`
3. 不要勾选 "Initialize this repository with a README"

然后执行：

```bash
# 添加远程仓库（替换 YOUR_USERNAME 和 YOUR_REPO）
git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

### 5. 启用 GitHub Pages

1. 进入你的 GitHub 仓库页面
2. 点击 **Settings** 标签
3. 在左侧菜单找到 **Pages**
4. 在 **Source** 下拉菜单中选择 **GitHub Actions**
5. 等待自动部署完成（通常需要 1-2 分钟）

### 6. 访问你的网站

部署完成后，你的网站地址将是：
```
https://YOUR_USERNAME.github.io/YOUR_REPO/
```

### 可能遇到的问题：

#### 问题 1: Main.zip 文件太大
如果 Main.zip 文件超过 100MB，Git 可能无法上传。

**解决方案**：
```bash
# 使用 Git LFS
git lfs install
git lfs track "*.zip"
git add .gitattributes
```

#### 问题 2: 路由刷新后 404
如果页面刷新后出现 404，需要添加 HashRouter。

**修改 `jiemian/src/app/routes.ts`**：
```typescript
import { createHashRouter } from "react-router";
import { Welcome } from "./pages/Welcome";
import { Galaxy } from "./pages/Galaxy";

export const router = createHashRouter([
  {
    path: "/",
    Component: Welcome,
  },
  {
    path: "/galaxy",
    Component: Galaxy,
  },
]);
```

#### 问题 3: 字体加载失败
Google Fonts 在某些地区可能无法访问。

**解决方案**：使用国内 CDN 或本地字体文件。
